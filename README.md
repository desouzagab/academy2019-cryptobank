## Como Utilizar:
### 1. Clone o repositório 
```
$ git clone https://github.com/desouzagab/academy2019-cryptobank.git
```
### 2. Entre na pasta criada 
```
$ cd academy2019-cryptobank
```
### 3. Instale as dependencias
```
yarn install
```
### 4. Rode o server
```
yarn serve
```
